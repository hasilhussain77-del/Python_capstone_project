 Python Data Analysis Capstone Project

## Overview

This project focuses on data analysis using Python libraries like Pandas and NumPy. It involves working with multiple datasets such as employee data, project data, and seniority levels to perform data cleaning, transformation, and analysis.

## Key Features

* Data cleaning and handling missing values
* Data transformation (splitting columns, formatting)
* Merging multiple datasets
* Applying business logic (bonus, promotion, demotion)
* Aggregation and filtering

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## How to Run

1. Install required libraries:
   pip install pandas numpy
2. Run the notebook or Python file
